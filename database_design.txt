### 2. Core Entities: User and Startup Lifecycle

#### **`users`**
*   **Purpose:** Stores user account information for authentication and identification.
| Column | Type | Constraints & Description |
| :--- | :--- | :--- |
| `id` | Integer | **Primary Key** |
| `email` | String(120) | Unique, Not Null, Indexed. User's login email. |
| `password_hash` | String(128) | Nullable. Hashed password. Null for OAuth users. |
| `full_name` | String(100) | Not Null. User's full name. |
| `is_verified` | Boolean | Default: `False`. If the user's email is verified. |
| `role` | Enum(UserRole) | Not Null, Default: `USER`. Role in the system (`USER`, `ADMIN`). |
| `created_at` | DateTime | Default: `utcnow`. Timestamp of account creation. |
| `google_id` | String(128) | Unique, Nullable. OAuth ID for Google. |
| `linkedin_id` | String(128) | Unique, Nullable. OAuth ID for LinkedIn. |

#### **`submissions`**
*   **Purpose:** Captures the initial application data from a prospective startup.
| Column | Type | Constraints & Description |
| :--- | :--- | :--- |
| `id` | Integer | **Primary Key** |
| `user_id` | Integer | **Foreign Key** to `users.id`. The submitting user. |
| `startup_name` | String(200) | The name of the proposed startup. |
| `founders_and_inspiration` | Text | Answer to the relevant application question. |
| `problem_statement` | Text | Answer to the relevant application question. |
| `who_experiences_problem` | Text | Answer to the relevant application question. |
| `product_service_idea` | Text | Answer to the relevant application question. |
| `how_solves_problem` | Text | Answer to the relevant application question. |
| `intended_users_customers` | Text | Answer to the relevant application question. |
| `main_competitors_alternatives` | Text | Answer to the relevant application question. |
| `how_stands_out` | Text | Answer to the relevant application question. |
| `startup_type` | String(100) | Answer to the relevant application question. |
| `status` | Enum(SubmissionStatus) | Not Null, Default: `PENDING`. The application's status. |
| `submitted_at` | DateTime | Default: `utcnow`. Timestamp of submission. |
| `raw_chat_data` | JSON | A full log of the application conversation for auditing. |

#### **`evaluations`**
*   **Purpose:** Stores the detailed, internal analysis of a submission.
| Column | Type | Constraints & Description |
| :--- | :--- | :--- |
| `id` | Integer | **Primary Key** |
| `submission_id` | Integer | **Foreign Key** to `submissions.id`, Unique. |
| `problem_analysis` | JSON | Structured analysis of the problem. |
| `solution_analysis` | JSON | Structured analysis of the solution. |
| `market_analysis` | JSON | Structured analysis of the market. |
| `growth_analysis` | JSON | Structured analysis of growth potential. |
| `competitor_analysis` | JSON | Structured analysis of competitors. |
| `risks_analysis` | JSON | Structured analysis of risks. |
| `overall_score` | Float | A final numerical score for the submission. |
| `final_decision` | String(100) | The final decision (e.g., "Approved", "Rejected"). |
| `overall_summary` | Text | A written summary of the evaluation. |
| `created_at` | DateTime | Default: `utcnow`. Timestamp of evaluation creation. |

#### **`startups`**
*   **Purpose:** The central entity representing an active startup in the platform.
| Column | Type | Constraints & Description |
| :--- | :--- | :--- |
| `id` | Integer | **Primary Key** |
| `user_id` | Integer | **Foreign Key** to `users.id`. The startup's owner. |
| `submission_id` | Integer | **Foreign Key** to `submissions.id`, Unique. The original application. |
| `name` | String(200) | Not Null. The official name of the startup. |
| `slug` | String(255) | Unique, Not Null, Indexed. A URL-friendly identifier. |
| `status` | Enum(StartupStatus) | Not Null, Default: `ACTIVE`. The startup's overall status. |
| `overall_progress` | Float | Default: `0.0`. A metric for overall progress. |
| `current_stage` | Enum(StartupStage) | Not Null, Default: `EVALUATION`. The startup's program stage. |
| `next_milestone` | String(255) | A user-defined next major milestone. |
| `recent_activity` | JSON | A log of recent activities for the dashboard feed. |
| `created_at` | DateTime | Default: `utcnow`. Timestamp of creation. |
| `updated_at` | DateTime | Default: `utcnow`, onupdate: `utcnow`. Timestamp of last update. |
